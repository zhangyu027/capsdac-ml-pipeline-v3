# GCP Productionization Plan

## Services
- Cloud Storage: raw, processed, model, and forecast artifact zones
- BigQuery: curated analytics tables and feature tables
- Vertex AI: training jobs, model registry, batch predictions, monitoring
- Cloud Build: CI/CD validation and deployment
- Artifact Registry: containers
- Cloud Composer or Workflows: orchestration

## Promotion path
1. Local portfolio demo
2. Cloud Storage + BigQuery batch pipeline
3. Vertex AI training and model registry
4. Batch prediction and monitoring
5. Dashboard/API serving
