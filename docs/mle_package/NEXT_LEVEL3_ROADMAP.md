# Level 3 Roadmap

To turn this into a stronger Senior MLE project, add:

1. XGBoost and LightGBM as optional model candidates.
2. MLflow experiment tracking and model registry.
3. SHAP or permutation-based explainability.
4. Great Expectations or Pandera data contracts.
5. Scheduled retraining with Airflow or Cloud Composer.
6. Drift alerting with thresholds and incident workflow.
7. Backtesting across multiple forecast horizons: 1, 3, and 6 months.
8. Prediction intervals or conformal uncertainty estimates.
9. Dockerized API with real model loading.
10. Feature store materialization and point-in-time joins.
